【解説】

作図・計測・地図切り替えツールを起動するためのボタンを地図上に配置します。 

// 作図ボタンを配置します.
// 地図上の左から400ピクセル、上から10ピクセルの位置にボタンを配置し、
// ボタンをクリックすると、地図上の左から100ピクセル、上から50ピクセルの位置に
// 作図ツールバーが表示されます。
webtis.createDrawingBtn(400, 10, 100, 50);

// 計測ボタンを配置します。
// 地図上の左から400ピクセル、上から10ピクセルの位置にボタンを配置し、
// ボタンをクリックすると、地図上の左から100ピクセル、上から50ピクセルの位置に
// 計測ツールバーが表示されます。
webtis.createMeasureBtn(400, 10, 100, 50);

// 地図切り替えボタンを配置します。
// 地図上の左から450ピクセル、上から10ピクセルの位置にボタンを配置します。
webtis.createMapChangerBtn(450, 10);
