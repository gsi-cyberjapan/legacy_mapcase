地図上のイベントを捕捉する

発生したイベント：

【解説】

webtis.event()を使うことで、地図上で発生したイベントを拾うことができます。 

// 地図上でのマウス左ボタンが押し下げられるのを捕捉します。
// 捕捉されると、onMouseDown関数が呼び出されます。
webtis.event("leftdown", "onMouseDown");

// 地図上でのマウス左ボタンが上がるのを捕捉します。
// 捕捉されると、onMouseUp関数が呼び出されます。
webtis.event("leftup", "onMouseUp");

// 地図の移動を捕捉します。
// 捕捉されると、onMapMove関数が呼び出されます。
webtis.event("mapmove", "onMapMove");

// 地図のズームレベル変更を捕捉します。
// 捕捉されると、onZoomChange関数が呼び出されます。
webtis.event("zoomchange", "onZoomChange");

// 地図上でのマウスのダブルクリックを捕捉します。
// 捕捉されると、onDblClick関数が呼び出されます。
webtis.event("dblclick", "onDblClick");
