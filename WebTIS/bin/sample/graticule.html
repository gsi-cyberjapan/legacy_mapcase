経緯度線を表示する

非表示
5度単位 （ズームレベル4以上で表示されます）
1度単位 （ズームレベル6以上で表示されます）
1分単位 （ズームレベル12以上で表示されます）
30秒単位 （ズームレベル13以上で表示されます）
15秒単位 （ズームレベルが14以上で表示されます）

【解説】

地図に経緯度線を表示するには、webtis.createGraticule()で経緯度線を表示する設定に切り替えた後、 webtis.setGraticuleInterval()で経緯度線の間隔および表示するズームレベルを設定します。 

// 経緯度線が表示されるようにします。
webtis.createGraticule();

// 経緯度線の間隔を5度単位に設定し、
// ズームレベル4以上で表示されるようにします。
webtis.setGraticuleInterval(5, 4);
